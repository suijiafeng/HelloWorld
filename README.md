# hello-world
vue2:  elemetnUI

vue3:  element-plus

一套代码，可以分别在vue2/vue3上面跑起来。

主要为了抹平vue2/vue3的差异，方便开发者开发同时支持vue2/vue3的项目。

什么时候考虑用到这个项目？

1、现有的组件库vue2版本向vue3版本过渡使用

2、想开发一个同时支持vue2/vue3新的组件库或者插件

3、工作不够饱和，就是想折腾一下
